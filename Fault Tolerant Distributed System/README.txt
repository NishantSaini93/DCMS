Steps to run the project:
1. Run the ORBD daemon on initial port 900 (ORBInitialPort 900).
2. Use heap space of 2GB in Eclipse IDE (Xmx 2048m).
3. Run the FrontEnd class file in FrontEnd package.
4. Run each Server file in Servers package.
5. Wait for front end to receive heart beat from primary.
6. Run the ManagerClient class in SystemUsers package and input an appropriate option in the console.
7. Run the TestCaseClient class file in TestCasesAndCheckers package to run the test cases.
8. Refresh the project to see the generated log files.
